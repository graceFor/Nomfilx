## Container - Presenter Pattern

- Container => data, state를 가지고, api 불러옴, 모든 로직 처리, Data
- Presenter => 그 데이터들을 보여주는 역할 (state 값 가지고 있지 않음, 그냥 함수형 컴포넌트만 존재), Style
